УСТАНОВКА:

1) Установите зависимости с помощью Composer (composer install)

2) В случае необходимости, установите права 777 на папки:
   /system/cache/
   /system/logs/
   /system/config/
   /uploads/ (на все подпапки)

3) Создайте базу данных и залейте в нее все SQL файлы, находящиеся в данной папке

4) Скопируйте файл Database.php.inst в /system/config/ переименовав его в Database.php и запишите в него реквизиты доступа к базе данных

5) Система готова к работе. LOGIN: admin PASSWORD: admin

ВНИМАНИЕ!
mobiCMS находитсяв стадии разработки и с очередными обновлениями могут прийти серьезные изменения в коде.

Если вы обновляетесь с репозитория, или переключились на другую ветку:

1) удалите из базы данных все таблицы и заново залейте все SQL файлы, находящиеся в данной папке.
2) полностью очистите папку /system/cache удалите все файлы и папки, что там находятся.
3) обновите зависимости с помощью Composer (composer update)
